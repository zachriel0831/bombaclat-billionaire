# Weekly Macro Line Brief

## Purpose
整理近一週全球政經、利率、資金流向與市場主線，不做新聞堆疊，而是提煉出真正影響未來 1–4 週投資決策的重點，並生成適合直接貼進 LINE 的精簡但有判斷力的摘要。

---

## Repository boundary
此 skill 只定義分析與輸出格式，不代表 Python 服務負責 LINE 觸達。Python 服務應把分析結果寫入 `t_market_analyses`；任何 LINE webhook、push 或使用者觸達都由下游 Java 系統負責。

---

## Primary goal
輸出一則「可快速閱讀、可直接採用、可轉傳」的每週市場摘要。核心目的不是列新聞，而是幫使用者完成以下判斷：

1. 看懂本週真正主線是什麼
2. 分辨市場目前偏風險擴張、震盪整理，還是轉向防守
3. 知道對美股的配置與操作含義
4. 知道對台股的配置與操作含義
5. 知道接下來應偏保守、偏中性、偏多但不追，或可逢回布局

---

## When to use
當使用者提出以下需求時使用：

- 整理這一週的世界局勢
- 本週全球經濟總結
- 幫我做一份每週市場摘要
- 最近國際局勢對美股、台股有何影響
- 給我 LINE 版的每週投資重點
- 幫我整理本週宏觀與市場重點
- 這週市場到底在反映什麼
- 幫我判斷下週美股、台股要偏多還是偏保守
- 幫我做一則可直接貼 LINE 的每週投資摘要

---

## Analysis window
- 預設觀察區間為近 7 天
- 若本週重大事件具延續性，可補充近 2–4 週背景，但主結論仍以近 7 天為主
- 若使用者未指定投資週期，預設以未來 1–4 週為決策窗口

---

## Default investment stance
若使用者未指定風格，預設採中性偏保守立場。
只有在多個核心訊號明確共振轉強時，才可提升為中性偏多或偏多但不追。

---

## Source priority
回答前，優先確認近一週是否有以下重大變化，並用它們來建構主線，不可平均分配篇幅：

### 1. 宏觀與政策
1. 美國聯準會與主要央行政策訊號
2. 美國就業、通膨、PMI、GDP、零售銷售、消費與企業活動數據
3. 美債殖利率曲線變化，至少看 2Y、10Y
4. 美元指數、油價、金價、主要原物料波動

### 2. 市場風險
1. 地緣政治衝突、制裁、軍事風險
2. 國際貿易、關稅、供應鏈與出口變化
3. 信用風險、流動性風險、能源與運輸風險

### 3. 科技與產業主線
1. AI、半導體、大型科技公司、資本支出動向
2. 雲端、ASIC、IP、先進製程、半導體設備、HPC/伺服器趨勢
3. 美股科技龍頭是否延續主導風險偏好

### 4. 台灣市場連動
1. 台灣出口數據與產業景氣
2. 外資流向、台指期方向、新台幣走勢
3. 台積電、AI 伺服器供應鏈、半導體設備、權值股表現
4. 金融股是否成為資金緩衝或防守區

## Evidence and web verification policy
- `t_relay_events`、`t_x_posts`、`t_market_index_snapshots`、`market_context:*` 事件是主要本地證據，但不是完整世界事件清單。
- 不可因本地資料缺席，就推論「沒有發生重大事件」。
- 若模型執行環境提供 web search，涉及最新政策、價格、戰事、總經數據、財報或突發事件時，必須查證最新資訊再下判斷。
- 若 web search 不可用，或查證後仍不足，應明確標示「資料缺口」或「待查證」，並降低語氣信心。
- 輸出時要區分三種來源：本地事件可見事實、外部查證事實、基於兩者形成的推論。
- 若本地資料與外部查證互相矛盾，優先說明矛盾點，不要強行合併成單一結論。

## Core analysis principles

### 1. 先抓主線，再談配置
輸出前先判斷本週市場的第一主線是什麼。
可能是：
- 利率預期變化
- 地緣政治與油價
- AI / 科技股估值擴張
- 景氣數據轉弱或轉強
- 美元與殖利率壓力
- 外資與匯率對亞洲股市的影響

若有多條線並存，必須排序，不可平均描述。

### 2. 先講影響，再講事件
不要先寫「發生了什麼」，而要先寫「市場因此怎麼反應、配置應怎麼調整」。

### 3. 市場強弱要分層
至少要隱含判斷以下層次：
- 主線是否明確
- 風險偏好是否擴散
- 是少數權值撐盤，還是全面轉強
- 是短線反彈，還是中期可延續的方向

### 4. 一定要給出操作等級
結論不可只有抽象形容，必須明確落在以下之一：
- 偏保守
- 中性偏保守
- 中性
- 中性偏多
- 偏多但不追
- 可逢回分批布局

### 5. 一定要區分「主線強」與「市場全面強」
若上漲主要由少數 mega cap 或權值股帶動，不可直接寫成全面轉強。
必須明確指出是：
- 權值主導
- 主線集中
- 廣度改善
- 防守輪動
- 結構仍偏保守

---

## Depth rule
- 預設不要只寫結論，需額外補足「原因 + 市場結構 + 操作含義」
- 每個段落除了結論，至少再補一句說明原因或條件
- 使用者讀完後，應能知道：
  1. 為什麼市場這樣走
  2. 這是全面轉強還是少數權值主導
  3. 接下來什麼條件下可以偏多，什麼條件下要轉保守

---

## Required reasoning layers
生成內容時，至少完成以下五層判讀：

### A. 本週主線判斷
用一句話指出最重要的市場驅動因子，並標示目前總體基調。

### B. 美股判讀
判斷本週美股屬於：
- 風險擴張
- 權值主導
- 科技強、廣度弱
- 防守輪動
- 利率壓抑估值
- 震盪整理

### C. 台股判讀
判斷台股目前屬於：
- 權值撐盤
- AI / 半導體主導
- 外資觀望
- 匯率壓力下震盪
- 高檔不追、逢回布局
- 防守輪動

### D. 風險排程
指出接下來一週最可能改變市場方向的變數，例如：
- CPI / PPI / 非農 / 零售銷售 / PMI
- FOMC / 官員談話
- 美債殖利率急升急跌
- 油價與地緣升溫
- 大型科技財報 / 指引
- 台積電法說 / 出口數據 / 外資轉向

### E. 操作含義
明確寫出配置建議，而非抽象描述。

1. 本週主線
2. 美股觀點
3. 台股觀點
4. 風險提醒
5. 一句結論

## Required content rules for each section

### 1. 本週主線
必須回答：
- 市場本週到底在交易什麼
- 是單一主線還是多線拉扯
- 目前總體基調偏風險擴張、震盪，或防守
- 若主線不只一條，哪一條是第一主線、哪一條只是干擾項
- 為什麼這條主線目前壓過其他因素

### 2. 美股觀點
必須回答：
- 美股偏多、震盪、還是偏保守
- 是大型科技主導，還是市場廣度同步改善
- 殖利率、美元、油價對估值是否形成壓力
- 若主線延續，哪些方向可續抱或逢回看；若主線轉弱，哪些方向要轉守
- 配置應偏大型科技、防守、公用事業、醫療、短債，或維持平衡

### 3. 台股觀點
必須回答：
- 台股下週偏權值撐盤、AI 主導、震盪整理，或防守
- 台積電、外資、台幣、成交量、出口與 AI 鏈是否支持偏多
- 是電子主流擴散，還是只有少數權值股撐盤
- 若主線仍在 AI / 半導體，要明確提醒是可逢回還是不宜追高
- 若訊號偏弱，要明確提醒偏保守、降低追價
- 若外資、台幣或量能轉弱，結構上會如何變化

### 4. 風險提醒
必須回答：
- 接下來一週最值得盯的 2–4 個風險或變數
- 這些變數若偏多 / 偏空，分別會如何影響市場
- 哪些事件最可能改變目前的市場基調

### 5. 一句結論
必須落在可執行的操作判斷，例如：
- 偏保守
- 中性偏保守
- 中性
- 偏多但不追
- 逢回分批布局
- 降低槓桿
- 偏大型權值與現金流穩健標的

- 優先使用短段落
- 每段 2–4 句為佳
- 避免單段過長
- 可使用全形冒號與換行
- 不使用表格
- 不堆疊過多項目符號

- 若下游要發送到 LINE，這裡只提供可讀格式，不處理觸達

## Writing style
- 風格像投資決策備忘錄，不像新聞摘要
- 優先講影響，再講事件
- 用字精簡、直接、有判斷
- 避免模糊形容，例如「值得持續關注」
- 若訊號矛盾，要直接寫出震盪偏保守或區間整理
- 若結構偏強但不適合追高，要明確寫出偏多但不追

- 預設輸出為標準完整版，控制在 500–700 字

## Decision rules

### Global interpretation
- 若戰事升溫、油價漲、殖利率升、美元走強：整體風險偏保守
- 若通膨降溫、降息預期升高、殖利率回落：成長股與科技股受惠
- 若就業轉弱但央行仍偏鷹：市場震盪機率高
- 若大型科技財報與資本支出強：AI 主線可延續，但須防估值過熱
- 若美元走強、亞洲貨幣偏弱：新興亞洲市場與外資流向壓力升高

### Market structure interpretation
- 若指數上漲主要由少數 mega cap 帶動：定義為權值主導，不可誤判為全面轉強
- 若科技股強，但中小型與廣度不佳：定義為主線集中，非全面風險擴張
- 若金融、防守、公用事業補漲：代表市場開始平衡或轉守
- 若殖利率與美元同步上行：高估值成長股壓力升高
- 若油價急升：留意通膨預期反彈與運輸、航空、塑化波動

### US stocks
- 偏防守時：大型權值、醫療、公用事業、短債優先
- 偏中性時：大型科技搭配防守資產
- 偏積極時：可提高 AI、半導體、雲端配置，但避免追高
- 若殖利率快速上行：壓抑高估值成長股
- 若科技股強但市場廣度弱：以龍頭為主，不擴大追逐二三線題材

### Taiwan stocks
- 若 AI、半導體、出口數據穩健：偏電子權值與 AI 鏈
- 若外資流出、台幣走弱、地緣風險升高：保守應對
- 若台積電鏈與伺服器鏈基本面仍強：逢回分批布局
- 若指數急漲且量價過熱：明確提醒不追高
- 若僅權值撐盤、中小型與 OTC 未跟：視為結構偏保守，不宜過度樂觀
- 若金融股轉強、電子休息：代表資金轉入防守或平衡配置

## Scenario fallback logic
若本週沒有單一主線，則改用以下邏輯：

1. 直接指出市場正同時受利率、地緣、科技估值拉扯
2. 將整體定調為：
   - 震盪偏保守
   - 區間整理
   - 中性偏保守
3. 建議採分批配置，不做單一方向重壓
4. 美股與台股都要明確區分可看但不追與應偏防守

## Example output (short)
本週主線：市場主軸仍是美國利率預期與 AI 科技股強勢，但油價與地緣風險讓整體風險偏好沒有全面擴散，屬於權值領漲下的震盪偏多。

美股：大型科技與半導體仍是主線，但若殖利率與美元續強，高估值題材容易震盪。配置上以龍頭科技搭配部分防守資產，不宜全面追高。

台股：台積電與 AI 伺服器鏈仍是核心觀察，若外資與台幣未明顯轉弱，逢回可分批看大型權值與主流供應鏈；若只是權值撐盤、中小型未跟，仍以中性偏保守看待。

風險提醒：下週需盯美國通膨與就業數據、殖利率是否再上行，以及油價與外資動向是否造成亞洲股市壓力。

一句結論：現階段可維持中性偏多，但只適合逢回布局主流龍頭，不宜追高擴張部位。

## Example output (full)
本週主線：市場核心仍是美國利率預期與 AI 科技股強勢，但油價與地緣風險同步升溫，讓風險偏好沒有全面擴散。換句話說，當前更像少數大型科技與半導體撐住指數，而不是全面性多頭重啟，因此整體定調仍是震盪中的中性偏多。利率與科技主線仍壓過其他因素，但油價與地緣風險已開始成為干擾市場情緒的第二條線。

美股：大型科技、半導體與 AI 資本支出主線暫時沒有被破壞，但結構上仍較接近權值主導，而非全面風險偏好擴散。若美債殖利率續升、美元轉強，高估值成長股波動會明顯放大，屆時二三線題材股壓力通常會大於龍頭股。操作上較適合以龍頭科技為核心，搭配部分防守配置；若後續市場廣度改善、非 mega cap 也能接棒，才代表風險偏好真正轉強。

台股：台積電、AI 伺服器鏈與先進製程供應鏈仍是盤面核心，但關鍵不只是 ADR 或美股科技股表現，更要看外資是否回補、台幣是否止穩，以及量能是否支持主流擴散。若只是台積電等權值股撐盤、中小型與 OTC 沒有同步轉強，台股仍應視為結構性偏保守，不宜把指數上漲直接解讀成全面回暖。反過來說，若權值穩、AI 鏈同步放量、外資現貨與期貨方向改善，才適合逢回分批布局主流龍頭；若外資、台幣或成交量再轉弱，則應把策略拉回中性偏保守。

風險提醒：下週需優先關注美國通膨與就業數據、殖利率是否再度上行，以及油價與中東局勢是否繼續推升避險情緒。若數據偏熱、殖利率續升，市場可能重新壓縮高估值科技股；若數據溫和、殖利率回落，則有利科技主線延續。對台股而言，外資現貨與期貨方向、新台幣走勢與台積電量價表現，會直接影響反彈的延續性。

一句結論：目前可維持中性偏多，但屬於主流龍頭可逢回布局、整體部位不宜躁進擴張的階段，策略上以選股與分批為主，不適合全面追高。
