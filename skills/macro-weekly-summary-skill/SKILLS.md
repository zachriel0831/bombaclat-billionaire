# Weekly Macro Line Brief

## Purpose
整理近一週全球政經與市場主線，提煉出真正影響投資決策的重點，並生成適合貼到 LINE 對話窗的精簡摘要。

## Primary goal
輸出一則「可快速閱讀、可直接採用、可轉傳」的每週市場摘要，不追求新聞羅列，而追求：
1. 看懂本週主線
2. 知道對美股有何影響
3. 知道對台股有何影響
4. 知道接下來應偏保守、偏中性，或可逢回布局

## When to use
當使用者提出以下需求時使用：
- 整理這一週的世界局勢
- 本週全球經濟總結
- 幫我做一份每週市場摘要
- 最近國際局勢對美股、台股有何影響
- 給我 LINE 版的每週投資重點

## Source priority
回答前，優先確認近一週是否有以下重大變化：
1. 地緣政治衝突、制裁、軍事風險
2. 美國聯準會與主要央行政策訊號
3. 美國就業、通膨、PMI、GDP、零售銷售等數據
4. 美債殖利率、美元、油價、金價、原物料波動
5. 國際貿易、關稅、供應鏈與出口變化
6. AI、半導體、大型科技公司與資本支出動向
7. 台灣出口、外資、新台幣、台積電、AI 伺服器供應鏈訊號

## Default investment stance
若使用者未指定風格，預設以「未來 1–4 週」為觀察窗口，採「中性偏保守」立場。

## Output length
預設輸出長度控制在 320–380 字。


## Required output structure
請依以下順序輸出：

1. 本週主線  
2. 美股觀點  
3. 台股觀點  
4. 風險提醒  
5. 一句結論

## Preferred formatting for LINE
- 優先使用短段落
- 每段 1–2 句
- 避免過長句
- 可使用全形冒號與換行
- 不使用表格
- 不堆疊過多項目符號
- 內容應可直接貼入 LINE 而不需二次編輯

## Writing style
- 使用繁體中文
- 風格像投資決策備忘錄，不像新聞稿
- 優先講影響，再講事件
- 用字精簡、直接、有判斷
- 結論需可執行，例如：
  - 偏防守
  - 分批布局
  - 逢回承接
  - 不宜追高
  - 降低槓桿
  - 偏大型權值與現金流穩健標的

## Decision rules

### Global interpretation
- 若戰事升溫、油價漲、殖利率升：整體風險偏保守
- 若通膨降溫、降息預期升高：成長股與科技股受惠
- 若就業轉弱但央行仍偏鷹：市場震盪機率高
- 若大型科技財報與資本支出強：AI 主線可延續，但仍須防估值過熱

### US stocks
- 偏防守時：大型權值、醫療、公用事業、短債優先
- 偏中性時：大型科技搭配防守資產
- 偏積極時：可提高 AI、半導體、雲端配置，但避免追高
- 若殖利率快速上行：壓抑高估值成長股

### Taiwan stocks
- 若 AI、半導體、出口數據穩健：偏電子權值與 AI 鏈
- 若外資流出、台幣走弱、地緣風險升高：保守應對
- 若台積電鏈與伺服器鏈基本面仍強：逢回分批布局
- 若指數急漲且量價過熱：明確提醒不追高

## Constraints
- 不可只列新聞
- 不可沒有美股與台股兩部分
- 不可只給模糊結論
- 不可寫成晨間新聞播報稿
- 若訊號矛盾，應直接寫出「震盪偏保守」或「區間整理」
- 涉及最新政策、價格、戰事、總經數據時，需先查證最新資訊

## Failure mode handling
若本週沒有單一主線，則改用以下邏輯：
- 指出市場正在同時受「利率、地緣、科技估值」拉扯
- 建議採中性偏保守、分批配置
- 避免單一方向重壓

## Output checklist
生成前請檢查：
- 是否有明確主線
- 是否有美股建議
- 是否有台股建議
- 是否有風險提醒
- 是否能直接貼到 LINE
- 是否控制在要求字數內

## Example output
本週主線：市場同時受中東風險、Fed 延後降息預期與科技股高估值拉扯，整體偏震盪。  
美股：配置上以大型科技搭配短債與防守型資產，避免追逐短線過熱題材。  
台股：AI 與半導體基本面仍撐盤，但高檔不宜追價，逢回分批看台積電鏈與伺服器族群。  
風險提醒：油價、殖利率與外資動向仍是短線變數。  
一句結論：現階段不是全面進攻，而是控風險下的選股與分批布局。
