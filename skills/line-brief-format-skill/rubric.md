# Rubric for Weekly Macro Summary

## Objective
讓輸出更像「投資決策摘要」，而不是「新聞整理」。

This rubric is for mobile-chat readability only. It does not imply Python owns LINE push, webhook, or user delivery.

## Scoring dimensions

### 1. 主線清晰度
- 是否能一句話講出本週最重要的市場主因
- 是否避免把不相關事件全部塞進去

### 2. 投資相關性
- 是否有明確連到美股與台股
- 是否有實際配置或操作含義

### 3. 精簡度
- 是否能在有限篇幅內保留有用資訊
- 是否刪除背景雜訊與新聞口吻

### 4. 可執行性
- 是否能讓讀者看完後知道應偏防守、偏中性，或逢回布局
- 是否避免只有空泛判斷

### 5. 手機聊天可讀性
- 是否分段清楚
- 是否句子不過長
- 是否適合手機閱讀與轉發

## Hard rules
- 必須有：主線 / 美股 / 台股 / 風險 / 結論
- 不可只有新聞摘要
- 不可省略市場影響
- 不可寫成過度學術或過度口語
- 預設 320–380 字最優先
